| Annotation    | Meaning                                             |
| ------------- |:---------------------------------------------------:|
| @Component    | generic stereotype for any Spring-managed component |
| @Repository   | stereotype for persistence layer                    |
| @Service      | stereotype for service layer                        |
| @Controller   | stereotype for presentation layer (spring-mvc)      |
